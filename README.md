# Rojo Malbec · App de Votación Expo

App de votación para la Expo Rojo Malbec (8, 9 y 10 de mayo 2026).

Los visitantes votan por su blend favorito y participan de un sorteo diario de **Glühwein**.

## Blends disponibles
- 🌿 Sal del Desierto
- ⚡ Nanami Tōgarashi  
- 🌑 Panch Phoron
- 🍒 Za'atar

## Cómo funciona
1. Los visitantes escanean el QR
2. Votan por su blend favorito
3. Dejan su número de teléfono
4. Participan del sorteo diario

## Admin
Acceder a `?admin=true` con contraseña para ver estadísticas y sortear ganadores.
